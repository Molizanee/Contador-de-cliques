# Contador-de-cliques
Um simples contador de cliques, desenvolvido para fins de estudo com o uso do kit de ferramentas do TkInter para construção de interfaces com Python!
# Para rodar o projeto é necessario fazer a instalação do customtkinter, para isso é necessario rodar o seguinte codigo no terminal: "pip install customtkinter", recomendo tambem que instale o tkinter dentro de um ambiente virtual como o venv (Para windows: py -m venv venv - Para ativar no windows: .\venv\Scripts\Activate.ps1 ou cmd) para evitar instalar o tkinter em outros projetos.
